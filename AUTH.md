# Auth & Permissions Matrix

## 1. Current State
There is no authentication or authorization in V1. All tools are fully public and ungated.

## 2. Future State (Phase 2+)
If accounts are introduced:
- **Anonymous User:** Can use all basic calculators.
- **Registered User:** Can save scenarios, generate PDFs, and access advanced calculator history.
- **Admin:** Can view aggregate usage analytics and manage global site content.
