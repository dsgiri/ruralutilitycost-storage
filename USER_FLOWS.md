# User Flows & Journey Maps

## 1. Primary User Journey: Discovery to Calculation
1. **Entry:** User lands on the Homepage (`/`).
2. **Discovery:** User identifies their required calculator from the Featured Tool or the Discovery Grid.
3. **Selection:** User clicks "View Tool" and navigates to a specific calculator page (e.g., `/tools/grain-bin-capacity`).
4. **Input:** User enters their parameters into the left/top input column.
5. **Real-time Feedback:** As inputs change, the right/bottom output column updates immediately with results.
6. **Action:** User takes the information for their own planning. (In the future, they might download a PDF or save the scenario).

## 2. Secondary User Journey: Trust & Context
1. **Entry:** User scrolls to the footer or top nav.
2. **Trust Check:** User clicks `/about` or `/legal` to understand the origin and disclaimers of the tools.
3. **Return:** User navigates back to the homepage or a tool via the top nav or footer links.

## 3. Pinned Tools (Favorites) Journey
1. **Pinning:** While on a tool page, user clicks the "Pin" or "Save to Favorites" button.
2. **Accessing:** User clicks the "Favorites" link in the navigation.
3. **Viewing:** User sees a grid of their pinned tools on the `/favorites` page and clicks one to return to it quickly.
